Starts a calibration which can be aborted with abort or finish by its own.

Start a calibration with the amount of points, specified screenSizes and the option of being a stepped calibration which requires to call next before each calibrated point. Returns a Stream with either a DataSuccess or a DataFailed message indicating the stream failed. In case of a DataSuccess message containing a CalibrationMessage.
