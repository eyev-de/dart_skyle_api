softDisconnect method Null safety

Future<void> softDisconnect()

Disconnects only the grpc client and makes it unusable.

All connections will throw a NotConnectedException. To reconnect the grpcs call trySoftReconnect

Implementation

Future<void> softDisconnect() async {
  try {
    logger?.i('Disconnecting active Skyle grpcs...');
    await calibration.abort();
    switchSettings.stop();
    await _channel?.terminate();
  } catch (e, st) {
    logger?.e('Skyle disconnecting clients failed:', e, st);
  } finally {
    _channel = null;
    _client = null;
    _setClients();
    _connection = Connection.connecting;
    _connectionStreamController.add(_connection);
    logger?.i('Disconnected Skyle grpcs...');
  }
}
