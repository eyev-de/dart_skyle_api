Stream<DataState<Switch>> start()

Starts a stream of Switch which can live until Skyle disconnects. Returns a Stream with either a DataSuccess or a DataFailed message indicating the stream failed. In case of a DataSuccess message containing a Switch indicating the current switch configuration.
